ટર્બો કન્સલ્ટન્સી સર્વિસીઝ

  •    ફક્ત વર્ષ ૨૦૧૩ અને બાદના વર્ષો માટે (વર્ષ ૨૦૧૧ અને ૨૦૨૨ માટે નહીં) શ્રીફળ (વર્ઝન ૧૩.૩૪) ઇન્સ્ટોલેશન ફાઈલ ડાઉનલોડ કરો.
    1. જો તમારા પ્રોગ્રામનું હાલનું વર્ઝન ૧૦.xx અથવા ૧૧.xx અથવા વધુ હોય તો જ ઉપર જ્ણાવેલ ફાઈલ ડાઉનલોડ કરો. અન્યથા માર્ગદર્શન માટે સંપર્ક કરો.
    2. તમારા પ્રોગ્રામનું હાલનું વર્ઝન જાણવા, પ્રોગ્રામ શરૂ કરો. તે ઉપરના ડાબા ખૂણે Ver અથવા Version બાદ દર્શાવવામાં આવ્યું છે.

    • ડાઉનલોડ અને ઇન્સ્ટોલ કરતાં પહેલાં...

      1. જો તમે તમારા વર્તમાન પેકેજ (જુના વર્ઝન)ને શરૂ કરવામાં કે ત્યાર બાદ વપરાશ દરમ્યાન કોઇ પણ મુશ્કેલીનો સામનો કરી રહ્યા હો તો સૌ પ્રથમ તેનું નિરાકરણ કરો અને ત્યાર બાદ જ નવું વર્ઝન ઇન્સ્ટોલ કરો.
      2. જો તમે એકથી વધુ ફર્મ ધરાવતા હો અને દરેક ફર્મનું ઓછામાં ઓછું એક વખત કેરી ફોર્વડ કરી લીધું હોય તો, વધુ કેરી ફોર્વડ કર્યા વગર, નવું વર્ઝન ઇન્સ્ટોલ કરી શકો છો. પરંતુ જો કોઇ ફર્મનું એક પણ વખત કેરી ફોર્વડ ન કર્યું હોય તો નવું વર્ઝન ઇન્સ્ટોલ કરતાં પહેલાં તે ફર્મનું કેરી ફોર્વડ જરૂરથી કરી લેવું.
      3. નવું વર્ઝન, ફક્ત વર્ષ ૨૦૧૩ (૦૧/૦૪/૨૦૧૩ થી ૩૧/૦૩/૨૦૧૪) થી ૨૦૨૨ (૦૧/૦૪/૨૦૨૨ થી ૩૧/૦૩/૨૦૨૩) માટે ઇન્સ્ટોલ કરવું. વર્ષ ૨૦૧૧ અને ૨૦૧૨ માટે આ વર્ઝન ઇન્સ્ટોલ કરવું નહીં. તે વર્ષો માટે નવું વર્ઝન અલગથી નીચે આપેલ છે.
      4. નવું વર્ઝન, ઇન્સ્ટોલ કરતાં પહેલા, સંબંધિત વર્ષનું બેક-અપ જરૂરથી લેવું.
      5. વર્તમાન વર્ષ તેમ જ અન્ય વર્ષના તમામ પ્રોગ્રામ અચુક બંધ કરી લેવા.
      6. અન્ય તમામ કોમ્પ્યુટરમાં પણ વર્તમાન વર્ષના તમામ પ્રોગ્રામ અચુક બંધ કરી લેવા.

    • સુધારાઓ :

      1. સુધારાઓની માહિતીની ફાઇલ જોવા અહીં ક્લિક કરો.

    • ડાઉનલોડ :

      1. ઇન્સ્ટોલેશન ફાઈલ (Install Shrifal in 2013.exe) ડાઉનલોડ કરો
      2. વિન્ડોઝ ૧૦ ના વપરાશકર્તાઓ માટે ઇન્સ્ટોલેશન ફાઈલ (Install Shrifal in 2013.exe) ડાઉનલોડ કરો

    • ઇન્સ્ટોલેશન માટેની સુચનાઓ :

      1. ઉપર જણાવેલ લિંક પર ક્લિક કરી ઇન્સ્ટોલેશન ફાઈલ ડેસ્કટોપ ઉપર ડાઉનલોડ કરો.
      2. ત્યાર બાદ ઇન્સ્ટોલેશન ફાઈલ ઉપર ડબલ ક્લિક કરતાં,સુધારાઓની માહિતીની અંગ્રેજી તેમ ગુજરાતી ફાઇલો ઉપરાંત અપડેશન માટે જરૂરી ફાઇલો ઉપલબ્ધ થશે અને જે વર્ષમાં નવું વર્ઝન ઇન્સ્ટોલ કરવું હોય તે વર્ષ દાખલ કરવાનું જણાવશે.
      3. હિસાબી વર્ષ સામે ૨૦૧૩ અથવા ૨૦૧૩ના વર્ષનો પ્રોગ્રામ શરૂ કરવા માટે સામાન્યપણે જે નામ જણાવતા હો, તે નામ દાખલ કરો અને એન્ટર દબાવો. આ રીતે જણાવ્યા પ્રમાણે કરતાં, વર્ષ ૨૦૧૩ માં વર્ઝન અપડેટ કરવાનું કાર્ય પુર્ણ થશે.
      4. હવે ૨૦૧૩ના વર્ષનો પ્રોગ્રામ શરૂ કરો અને તમામ ફર્મ માટે ઇન્ડેક્ષીંગ કરી લો.
      5. આ જ રીતે વર્ષ ૨૦૧૪માં નવું વર્ઝન ઇન્સ્ટોલ કરવા, ઊપર જણાવેલ સુચના નંબર ૨ થી ૪ માં જણાવ્યા પ્રમાણે કરવું. હિસાબી વર્ષ સામે ૨૦૧૪ અથવા ૨૦૧૪ના વર્ષનો પ્રોગ્રામ શરૂ કરવા માટે સામાન્યપણે જે નામ જણાવતા હો, તે નામ દાખલ કરવું. હિસાબી વર્ષ ૨૦૧૫ થી ૨૦૨૧ માટે પણ આ જ પ્રમાણે કરવું.
      6. આ રીતે નવું વર્ઝન સફળતાપુર્વક ઇન્સ્ટોલ કરી લીધા બાદ, ઇન્સ્ટોલેશન ફાઇલોની જરૂરિયાત ન રહેતાં, તેમને કાઢી નાખવા "Delete unwanted Shrifal Installation Files for 2013" ફાઇલ ઉપર ડબલ ક્લિક કરો.
      7. સુધારાઓની વિસ્તૃત માહિતી, Shrifal Changes (Guj) in 2013.pdf ફાઇલમાં આપવામાં આવી છે.

  •    ફક્ત વર્ષ ૨૦૧૧ અને ૨૦૧૨ માટે (વર્ષ ૨૦૧૩ અને બાદના વર્ષો માટે નહીં) શ્રીફળ (વર્ઝન ૧૧.૧૮) ઇન્સ્ટોલેશન ફાઈલ ડાઉનલોડ કરો.
    1. જો તમારા પ્રોગ્રામનું હાલનું વર્ઝન ૧૦.xx અથવા ૧૧.xx અથવા વધુ હોય તો જ ઉપર જ્ણાવેલ ફાઈલ ડાઉનલોડ કરો. અન્યથા માર્ગદર્શન માટે સંપર્ક કરો.
    2. તમારા પ્રોગ્રામનું હાલનું વર્ઝન જાણવા, પ્રોગ્રામ શરૂ કરો. તે ઉપરના ડાબા ખૂણે Ver અથવા Version બાદ દર્શાવવામાં આવ્યું છે.

    • ડાઉનલોડ અને ઇન્સ્ટોલ કરતાં પહેલાં...

      1. જો તમે તમારા વર્તમાન પેકેજ (જુના વર્ઝન)ને શરૂ કરવામાં કે ત્યાર બાદ વપરાશ દરમ્યાન કોઇ પણ મુશ્કેલીનો સામનો કરી રહ્યા હો તો સૌ પ્રથમ તેનું નિરાકરણ કરો અને ત્યાર બાદ જ નવું વર્ઝન ઇન્સ્ટોલ કરો.
      2. જો તમે એકથી વધુ ફર્મ ધરાવતા હો અને દરેક ફર્મનું ઓછામાં ઓછું એક વખત કેરી ફોર્વડ કરી લીધું હોય તો, વધુ કેરી ફોર્વડ કર્યા વગર, નવું વર્ઝન ઇન્સ્ટોલ કરી શકો છો. પરંતુ જો કોઇ ફર્મનું એક પણ વખત કેરી ફોર્વડ ન કર્યું હોય તો નવું વર્ઝન ઇન્સ્ટોલ કરતાં પહેલાં તે ફર્મનું કેરી ફોર્વડ જરૂરથી કરી લેવું.
      3. નવું વર્ઝન, ફક્ત વર્ષ ૨૦૧૧ (૦૧/૦૪/૨૦૧૧ થી ૩૧/૦૩/૨૦૧૨) અને ૨૦૧૨ (૦૧/૦૪/૨૦૧૨ થી ૩૧/૦૩/૨૦૧૩) માટે ઇન્સ્ટોલ કરવું. વર્ષ ૨૦૧૩ અને ૨૦૧૪ માટે આ વર્ઝન ઇન્સ્ટોલ કરવું નહીં. તે વર્ષો માટે નવું વર્ઝન અલગથી ઉપર આપેલ છે.
      4. નવું વર્ઝન, ઇન્સ્ટોલ કરતાં પહેલા, બન્ને વર્ષનું બેક-અપ જરૂરથી લેવું.
      5. વર્તમાન વર્ષ તેમ જ અન્ય વર્ષના તમામ પ્રોગ્રામ અચુક બંધ કરી લેવા.
      6. અન્ય તમામ કોમ્પ્યુટરમાં પણ વર્તમાન વર્ષના તમામ પ્રોગ્રામ અચુક બંધ કરી લેવા.

    • ડાઉનલોડ :

      1. સુધારાઓની માહિતીની ફાઇલ (Shrifal Changes (Guj) in 2011.pdf) ડાઉનલોડ કરો
      2. ઇન્સ્ટોલેશન ફાઈલ (Install Shrifal in 2011.exe) ડાઉનલોડ કરો

    • ઇન્સ્ટોલેશન માટેની સુચનાઓ :

      1. ઊપર જણાવેલ બન્ને ફાઇલો ડેસ્કટોપ ઉપર ડાઉનલોડ કરો.
      2. ત્યાર બાદ ઇન્સ્ટોલેશન ફાઈલ ઉપર ડબલ ક્લિક કરતાં, અપડેશન માટે જરૂરી ફાઇલો ઉપલબ્ધ થશે અને જે વર્ષમાં નવું વર્ઝન ઇન્સ્ટોલ કરવું હોય તે વર્ષ દાખલ કરવાનું જણાવશે.
      3. હિસાબી વર્ષ સામે ૨૦૧૧ અથવા ૨૦૧૧ના વર્ષનો પ્રોગ્રામ શરૂ કરવા માટે સામાન્યપણે જે નામ જણાવતા હો, તે નામ દાખલ કરો અને એન્ટર દબાવો. આ રીતે જણાવ્યા પ્રમાણે કરતાં, વર્ષ ૨૦૧૧ માં વર્ઝન અપડેટ કરવાનું કાર્ય પુર્ણ થશે.
      4. હવે ૨૦૧૧ના વર્ષનો પ્રોગ્રામ શરૂ કરો અને તમામ ફર્મ માટે ઇન્ડેક્ષીંગ કરી લો.
      5. આ જ રીતે વર્ષ ૨૦૧૨માં નવું વર્ઝન ઇન્સ્ટોલ કરવા, ઊપર જણાવેલ સુચના નંબર ૨ થી ૪ માં જણાવ્યા પ્રમાણે કરવું. હિસાબી વર્ષ સામે ૨૦૧૨ અથવા ૨૦૧૨ના વર્ષનો પ્રોગ્રામ શરૂ કરવા માટે સામાન્યપણે જે નામ જણાવતા હો, તે નામ દાખલ કરવું.
      6. આ રીતે નવું વર્ઝન બન્ને વર્ષમાં સફળતાપુર્વક ઇન્સ્ટોલ કરી લીધા બાદ, ઇન્સ્ટોલેશન ફાઇલોની (Shrifal Changes (Guj) in 2011.pdf સિવાયની) જરૂરિયાત ન રહેતાં, તેમને કાઢી નાખવા "Delete unwanted Shrifal Installation Files for 2011" ફાઇલ ઉપર ડબલ ક્લિક કરો.
      7. સુધારાઓની વિસ્તૃત માહિતી, Shrifal Changes (Guj) in 2011.pdf ફાઇલમાં આપવામાં આવી છે.

ફાઈલ કોપી કરવા, નીચે જણાવ્યા પ્રમાણે કરો.

  1. જ્યાંથી ફાઈલ કોપી કરવી હોય તે સ્થળ (location) પર જાઓ.
  2. કી બોર્ડ દ્વારા: જે ફાઈલ કોપી કરવી હોય તે સિલેક્ટ (પસંદ) કરો. ફાઈલ સિલેક્ટ કરવા એરો કીનો ઉપયોગ કરો અથવા તે ફાઈલના શરૂઆતના થોડા અક્ષરો ઝડપથી ટાઇપ કરો અથવા તે ફાઈલનો પહેલો અક્ષર ધીમે ધીમે એટલી વાર દબાવો જેથી તે ફાઈલ સિલેક્ટ થઇ જાય. ત્યાર બાદ Ctrl + C દબાવતાં તે ફાઈલ કોપી થઇ જશે.
    માઉસ દ્વારા: તે ફાઇલ ઉપર રાઇટ-ક્લિક કરો (માઉસનું જમણું બટન દબાવો) અને કોપી (Copy) વિકલ્પ ઉપર ક્લિક કરો.

ફાઈલ પેસ્ટ કરવા, નીચે જણાવ્યા પ્રમાણે કરો.

  1. જે સ્થળે ફાઈલ પેસ્ટ કરવી હોય તે સ્થળ (location) પર જાઓ.
  2. કી બોર્ડ દ્વારા: Ctrl + V દબાવીને ફાઇલ પેસ્ટ કરો.
    માઉસ દ્વારા: ખાલી જગ્યા ઉપર માઉસનું પોઇન્ટર રાખી રાઇટ-ક્લિક કરો અને પેસ્ટ (Paste) વિકલ્પ ઉપર ક્લિક કરો.
